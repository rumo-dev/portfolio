HDRレンダリングされたシーンの高輝度部分を光源のようににじませて表現するため、AtmosFioreでは **Bloom** を実装しています。単純なぼかしではなく、複数解像度でのダウンサンプル・アップサンプルを組み合わせることで、広がりのある自然な発光を低コストで再現しています。

---

## 📐 Bloomの基本概念

Bloomは「輝度抽出 → 複数段階のガウシアンブラー → 合成」という3段構成のポストエフェクトです。単一解像度でブラー半径を大きくすると処理コストが爆発的に増えるため、AtmosFioreではミップチェインのように段階的に解像度を落としながらブラーをかける手法を採用しています。

### 処理フロー

1. **輝度抽出（Glow Extraction）**：しきい値を超えたピクセルのみを抽出
2. **ダウンサンプリング + ガウシアンブラー**：6段階の解像度でピンポンブラーを実行
3. **アップサンプリング + 合成**：全段の結果を加算合成
4. **最終合成**：元のHDR画像にBloom結果を加算

## 💻 C++実装

```cpp
/**
 * @brief Bloom（ブルーム）ポストエフェクトクラス
 *
 * @details
 * 高輝度部分を抽出し、ガウシアンブラーを複数段階で適用して
 * 元画像に合成することで発光表現を実現する。
 *
 * 処理フロー：
 * 1. 輝度抽出（threshold）
 * 2. ダウンサンプリング + ブラー
 * 3. アップサンプリング + 合成
 * 4. 最終合成
 *
 * @note
 * ・ピンポンバッファ構造（[N][2]）でブラーを効率化
 * ・downsample段数を固定することで安定したパフォーマンス
 * ・Fullscreen Quad を使ったポストプロセス前提設計
 */
class bloom
{
public:
	bloom(ID3D11Device* device, uint32_t width, uint32_t height);

	void make(ID3D11DeviceContext* immediate_context, ID3D11ShaderResourceView* color_map);

	ID3D11ShaderResourceView* getColorMap() const { return bloom_final->GetColorMap(); }

public:
	int   is_bloom = 1;                          ///< Bloom有効フラグ
	float bloom_extraction_threshold = 1.f;       ///< 輝度抽出しきい値
	float bloom_intensity = 0.673f;               ///< Bloom強度

private:
	std::unique_ptr<FullscreenQuad> bit_block_transfer;
	std::unique_ptr<Framebuffer> glow_extraction;   ///< 輝度抽出用バッファ

	static const size_t downsampled_count = 6;      ///< ダウンサンプル段数
	std::unique_ptr<Framebuffer> gaussian_blur[downsampled_count][2]; ///< ピンポンバッファ

	std::unique_ptr<Framebuffer> bloom_final;       ///< 最終出力

	Microsoft::WRL::ComPtr<ID3D11PixelShader> glow_extraction_ps;
	Microsoft::WRL::ComPtr<ID3D11PixelShader> gaussian_blur_downsampling_ps;
	Microsoft::WRL::ComPtr<ID3D11PixelShader> gaussian_blur_horizontal_ps;
	Microsoft::WRL::ComPtr<ID3D11PixelShader> gaussian_blur_vertical_ps;
	Microsoft::WRL::ComPtr<ID3D11PixelShader> gaussian_blur_upsampling_ps;
	Microsoft::WRL::ComPtr<ID3D11PixelShader> bloom_final_ps;

	struct alignas(16) bloom_constants
	{
		int   is_bloom;
		float bloom_extraction_threshold;
		float bloom_intensity;
		float something;
	};
	Microsoft::WRL::ComPtr<ID3D11Buffer> constant_buffer;
};
```

### 実行部分（`make`）の要点

```cpp
void bloom::make(ID3D11DeviceContext* immediate_context, ID3D11ShaderResourceView* color_map)
{
	// 1. 輝度抽出
	glow_extraction->Activate(immediate_context);
	bit_block_transfer->Blit(immediate_context, &color_map, 0, 1, glow_extraction_ps.Get());
	glow_extraction->Deactivate(immediate_context);

	// 2. ダウンサンプリング + ピンポンブラー（6段階ループ）
	// http://rastergrid.com/blog/2010/09/efficient-gaussian-blur-with-linear-sampling/
	for (size_t i = 1; i < downsampled_count; ++i)
	{
		// downsample → horizontal blur → vertical blur を段階ごとに繰り返す
		bit_block_transfer->Blit(immediate_context, gaussian_blur[i - 1][0]->GetColorMapAddress(),
			0, 1, gaussian_blur_downsampling_ps.Get());
		bit_block_transfer->Blit(immediate_context, gaussian_blur[i][0]->GetColorMapAddress(),
			0, 1, gaussian_blur_horizontal_ps.Get());
		bit_block_transfer->Blit(immediate_context, gaussian_blur[i][1]->GetColorMapAddress(),
			0, 1, gaussian_blur_vertical_ps.Get());
	}

	// 3. 全段を1回のドローで加算アップサンプリング
	std::vector<ID3D11ShaderResourceView*> srvs;
	for (size_t i = 0; i < downsampled_count; ++i)
		srvs.push_back(gaussian_blur[i][0]->GetColorMap());
	bit_block_transfer->Blit(immediate_context, srvs.data(), 0, downsampled_count, gaussian_blur_upsampling_ps.Get());

	// 4. 最終合成
	ID3D11ShaderResourceView* composite[2]{ color_map, glow_extraction->GetColorMap() };
	bit_block_transfer->Blit(immediate_context, composite, 0, 2, bloom_final_ps.Get());
	bloom_final->GenerateMips(immediate_context);
}
```

## 💻 HLSL実装

### 輝度抽出

```hlsl
// glow_extraction_ps.hlsl
Texture2D hdr_color_buffer_texture : register(t0);
float4 main(float4 position : SV_POSITION, float2 texcoord : TEXCOORD) : SV_TARGET
{
    float4 sampled_color = hdr_color_buffer_texture.Sample(sampler_states[POINT_CLAMP], texcoord);
    sampled_color = clamp(sampled_color, 0.0, 1.0);

    return float4(step(bloom_extraction_threshold, max(sampled_color.r, max(sampled_color.g, sampled_color.b)))
                  * sampled_color.rgb, sampled_color.a);
}
```

### ガウシアンブラー（線形サンプリングによる高速化）

```hlsl
// gaussian_blur_horizontal_ps.hlsl
Texture2D hdr_color_buffer_texture : register(t0);

float4 main(float4 position : SV_POSITION, float2 texcoord : TEXCOORD) : SV_TARGET
{
    uint mip_level = 0, width, height, number_of_levels;
    hdr_color_buffer_texture.GetDimensions(mip_level, width, height, number_of_levels);

    // http://rastergrid.com/blog/2010/09/efficient-gaussian-blur-with-linear-sampling/
    const float offset[3] = { 0.0, 1.3846153846, 3.2307692308 };
    const float weight[3] = { 0.2270270270, 0.3162162162, 0.0702702703 };

    float4 sampled_color = hdr_color_buffer_texture.Sample(sampler_states[LINEAR_CLAMP], texcoord) * weight[0];
    for (int i = 1; i < 3; i++)
    {
        sampled_color += hdr_color_buffer_texture.Sample(sampler_states[LINEAR_CLAMP], texcoord + float2(offset[i] / width, 0.0)) * weight[i];
        sampled_color += hdr_color_buffer_texture.Sample(sampler_states[LINEAR_CLAMP], texcoord - float2(offset[i] / width, 0.0)) * weight[i];
    }
    return sampled_color;
}
```

### アップサンプリング合成 & 最終合成

```hlsl
// gaussian_blur_upsampling_ps.hlsl
static const uint downsampled_count = 6;
Texture2D downsampled_textures[downsampled_count] : register(t0);

float4 main(float4 position : SV_POSITION, float2 texcoord : TEXCOORD) : SV_TARGET
{
    if (!is_enabled)
        return downsampled_textures[0].Sample(sampler_states[LINEAR_CLAMP], texcoord);

    float3 sampled_color = 0;
    [unroll]
    for (uint i = 0; i < downsampled_count; ++i)
        sampled_color += downsampled_textures[i].Sample(sampler_states[LINEAR_CLAMP], texcoord).xyz;

    return float4(sampled_color * bloom_intensity, 1);
}

// bloomfinal.hlsl
Texture2D texture_maps[2] : register(t0);
float4 main(VS_OUT pin) : SV_TARGET
{
    float4 color = texture_maps[0].Sample(sampler_states[POINT_CLAMP], pin.texcoord);
    float4 bloom = texture_maps[1].Sample(sampler_states[POINT_CLAMP], pin.texcoord);
    return float4(color.rgb + bloom.rgb, color.a);
}
```

## 🎨 処理の詳細

### 1. 6段階のミップチェイン構造

`downsampled_count = 6` により、フル解像度から1/64解像度まで段階的にブラーをかけます。低解像度側ほど大きな半径のボケを表現でき、高解像度側は輪郭の鋭いにじみを担当します。

### 2. Linear Sampling最適化

隣接する2テクセルを1回のサンプリングで平均化するテクニックにより、通常より少ないサンプル数で同等品質のガウシアンブラーを実現しています。

### 3. ミップマップ生成による後段連携

```cpp
bloom_final->GenerateMips(immediate_context);
```

最終出力に対してフルミップチェインを生成しておくことで、後段のAdaptation（輝度計算）が最上位ミップ（1x1）を平均輝度として参照できます。

## 🔧 実装のポイント

### 1. ピンポンバッファによるメモリ効率化

`gaussian_blur[downsampled_count][2]` の2枚1組バッファを使い回すことで、水平・垂直ブラーを交互に適用しながら余分なメモリ確保を避けています。

### 2. しきい値ベースの抽出

`step(bloom_extraction_threshold, ...)` により、しきい値未満のピクセルは完全に0として扱われるため、Bloomは「明るい部分だけ」に選択的に発生します。

<Aside type="note">
  Bloomは他のポストエフェクトと同様にFullscreen Quadを使用したポストプロセス前提設計です。
</Aside>

<Aside type="caution">
  入力SRV（`color_map`）は処理中にRTVとしてバインドしてはならず、同一リソースのSRV/RTV同時使用はDirectX 11の未定義動作となります。
</Aside>
